# Results Directory

This directory will contain output files from the analysis scripts, including:
- `growth_correlations.csv` – Spearman correlations between bacterial taxa and growth z‑scores
- `significant_weight_taxa.csv` – Taxa significantly correlated with weight‑for‑age
- `significant_height_taxa.csv` – Taxa significantly correlated with height‑for‑age
- `*.png` – Visualization plots

## Usage
The analysis scripts automatically create these files when run.
